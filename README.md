# 內容
## 角色劃分：
### 管理者分為兩種角色：館長和館員。
## 權限與決策：
  館長有權決定圖書館的休息時間和館員的月薪。

  館員負責執行日常管理任務，但是不具備調整休息時間或者自行調整薪資的權限。
  
## 管理者任務
### 館長任務：
  設定圖書館的休息時間。
  
  設定館員的月薪。
### 館員任務：
  圖書館日常管理任務，借還書的處理、圖書館內部維護。

## 借閱者功能：
### 角色劃分：
  借閱者分為會員和非會員。
### 功能區分：
  會員可能享有一些額外的特權，如更長的借閱期限或更多的借閱數量。
### 查閱圖書功能：
  非會員也可以使用系統查詢圖書館的藏書。
