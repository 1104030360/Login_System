# Usage
1.啟動 Django 開發伺服器：
 ```sh
python manage.py runserver
```
2.瀏覽器中打開 http://localhost:8000/ 以訪問應用程式。

3.您可以通過在導航欄中點擊 "註冊" 鏈接來註冊新使用者。

4.註冊後，您可以使用導航欄中的 "登入" 鏈接登入。

5.登入後，您將可以看到一個包含您的名字的歡迎訊息。

6.如果要登出，請在導覽欄中點擊 "登出" 鏈接。

# Dependencies
* Django 4.2.11
* Bootstrap 4.6.2
* jQuery 3.5.1
